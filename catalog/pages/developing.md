### Install dependancies

``` code
lang: shell
---
yarn
```

### Build library

``` code
lang: shell
---
yarn build
```

### Build and watch for changes

``` code
lang: shell
---
yarn build --watch
```

### Build documentation and examples

``` code
lang: shell
---
yarn build-catalog
```

### Start hot reloading documentation

``` code
lang: shell
---
yarn start
```

### Run tests

``` code
lang: shell
---
yarn test
```
